This plugin adds a mailing list subscribe form to your blog that syncs with Direct Mail for OS X.

== Description ==

If you use [Direct Mail for Mac OS X](http://directmailmac.com), this widget places a small subscribe form on your blog that lets readers quickly and easily add themselves to any [subscribe form](http://directmailmac.com/support/article/225) that you have set up inside of Direct Mail.

== Installation ==

1. Upload `DirectMailSubscribeForm.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the 'Appearance > Widgets' menu in WordPress to place the widget on your blog.
